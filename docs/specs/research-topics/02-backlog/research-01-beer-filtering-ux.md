# Research: Beer Filtering & Sorting UX

## Summary

The beer collection is a flat grid with no way to filter or sort. Research the best UX patterns for filtering a small-to-medium catalogue (10–100 items) in a Next.js app. Consider: sidebar filters vs inline chips vs search bar, multi-select vs single-select, and URL-synced state for shareable filtered views.

## Questions

- [ ] What filtering patterns work best for a card-grid catalogue? (sidebar, toolbar chips, dropdown, search)
- [ ] Should filter state live in URL search params, React state, or both?
- [ ] How should sorting interact with filters? (separate control vs combined)
