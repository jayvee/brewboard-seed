# Research: Beer Detail Page Design

## Summary

Each beer is currently a card with no detail view. Research how to add a detail page or expandable panel for individual beers. Consider: dedicated `/beer/[id]` route vs modal/sheet overlay vs expandable card, what content to show (tasting notes, ABV, brewery info, similar beers), and image placeholder strategies.

## Questions

- [ ] What navigation pattern fits best? (route-based page, modal overlay, or inline expand)
- [ ] What data fields should the detail view include beyond what the card shows?
- [ ] How should "similar beers" or "more from this brewery" recommendations be structured?
